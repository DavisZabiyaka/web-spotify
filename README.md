# web-spotify
Create web-application mimicking Spotify mobile application using MVC frameworks (Spring and AngularJS)

Standard for web applications
/main
  |-> /java
       | BootStart.java
       |-> /security
              |SecurityConfiguration.java
  |-> /resources
         |-> /static
               |-> /css /** bootstrap location */
               |-> /js
               |-> /fonts
         |-> /templates
               | /user
                   | sample.html
